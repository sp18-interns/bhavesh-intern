# SQL

## Weather Observation Station 6

## Problem Statement : 


- Query the list of CITY names starting with vowels (i.e., a, e, i, o, or u) from STATION. Your result cannot contain duplicates.

- Solution :

    SELECT DISTINCT CITY
    FROM STATION
    WHERE CITY RLIKE '^[aeiou]'
